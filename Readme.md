Blogmarks 3
===========

Get Started
-----------

    git clone git@bitbucket.org:znarf/blogmarks3.git
    cd blogmarks3
    git submodule update --init

    curl -s https://getcomposer.org/installer | php
    php composer.phar update

    cd public
    npm install -g bower
    bower install

    php -S localhost:8002
